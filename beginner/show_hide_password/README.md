# Project: Custom Password Input 

## Description:
A typescript/ Vanilla Javascript custom password Input.
Build with a view towards exploring how the workings of the password input element may work.

## Getting started:
Clone the repo and run
```bash
npm i
```
or
```bash
npm install
```
to install the needed packages.
NB: Remember you need node on your system.
